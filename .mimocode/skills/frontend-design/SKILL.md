# Frontend Design Skill

> Guidance for distinctive, intentional visual design when building new UI.

## Design Principles

### 1. Ground it in the Subject
- Name one concrete subject, its audience, and the page's single job
- Use the subject's own world, materials, and vernacular
- Build with the brief's real content throughout

### 2. Hero as Thesis
- Open with the most characteristic thing in the subject's world
- Be deliberate: headline, image, animation, live demo, interactive moment
- Avoid template answers (big number + small label + gradient accent)

### 3. Typography as Personality
- Pair display and body faces deliberately
- Set clear type scale with intentional weights, widths, spacing
- Make type treatment memorable, not neutral

### 4. Structure is Information
- Numbering, eyebrows, dividers, labels encode truth
- Don't decorate - encode
- Question numbered markers: only use if content is actually sequential

### 5. Deliberate Motion
- Where and if animation serves the subject
- Page-load sequence, scroll-triggered reveal, hover micro-interactions
- Orchestrated moment > scattered effects
- Less is more - extra animation feels AI-generated

### 6. Match Complexity to Vision
- Maximalist = elaborate execution
- Minimalist = precision in spacing, type, detail
- Elegance = executing the chosen vision well

### 7. Intentional Copy
- Words are design material, not decoration
- Write from user's side of screen
- Name things by what people control, not how system is built
- Use active voice: "Save changes" not "Submit"

## Process

1. **Brainstorm**: Create token system (color, type, layout, signature)
2. **Review**: Check against brief, revise defaults
3. **Build**: Follow revised plan exactly
4. **Critique**: Self-review, take screenshots, remove one accessory

## Writing in Design

- Plain verbs, sentence case, no filler
- Tone matched to brand and audience
- Each element does exactly one job
- Errors explain what went wrong, not apologize
- Empty screens are invitations to act

## Anti-Patterns to Avoid

1. Warm cream + serif + terracotta (AI default)
2. Near-black + acid-green accent (AI default)
3. Broadsheet layout with hairline rules (AI default)

## Checklist

- [ ] Subject grounded in real content
- [ ] Hero is a thesis, not decoration
- [ ] Typography is memorable
- [ ] Structure encodes information
- [ ] Motion is deliberate
- [ ] Copy is intentional
- [ ] One bold choice, rest disciplined
- [ ] Responsive to mobile
- [ ] Keyboard focus visible
- [ ] Reduced motion respected
